## 💥 Troubleshooting

### Build Issues
- If build fails, check for missing npm dependencies
- Run: `npm install aws-amplify @aws-amplify/api`

### Runtime Issues
- Check browser console for error messages
- Look for "❌" prefixed messages in console
- Verify AWS credentials in amplify_outputs.json
- Check AppSync API logs in AWS Console

### Data Issues
- Verify backend schema matches expected models
- Check DynamoDB tables in AWS Console
- Review AppSync resolvers for errors
